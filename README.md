# U.S. Soybean Export Markets Visualization (Tkinter + Matplotlib)

## Overview
This project uses **Python**, **Tkinter**, and **Matplotlib** to visualize the **Top 10 U.S. soybean export markets by volume** based on the provided data.

The application displays the charts in **two tabs**:
- **Oct 2024**
- **Oct 2025**

> Note: The provided dataset contains **January–October 2024** and **January–October 2025** soybean export data.  
> Since separate November data was not included, the tabs are implemented using the available provided data.
